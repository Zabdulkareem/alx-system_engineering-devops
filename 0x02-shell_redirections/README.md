This repository consists of explanations to the script used for Shell I/o Redirections and filters

- 0. ***cat "Hello, World"*** prints "Hello, World" to the standard output.
- 1. ***echo \"\(\Ôo\)\'*** will output a confused smiley to the console.
- 2. ***cat file_name*** will display content of the file ``'file_name'``.
- 3. ***cat fileA fileB*** concatenates the contents of fileA and fileB, displayed to the stdout.
- 4. ***tail file_name*** will display the last ten lines of the file ``file_name``.
- 5. ***head file_name*** outputs the first ten lines of the file ``file_name``.
- 6. ***head -3 file_name | tail +3*** displays the  3rd line in the file.
- 7. ***echo "Best" > file_name*** writes the text "Best" into the file.
